# Proyecto Testores y Matrices Básicas (BT – Boolean Testor Algorithm)

Este proyecto implementa un conjunto de herramientas para operar con **matrices booleanas**, **testores**, **testores típicos**, **matrices básicas** y diferentes **operadores combinatorios**, basados en el algoritmo BT y en el análisis de matrices binarias estructuradas.

El sistema automatiza completamente el flujo:

1. **Generación de una matriz booleana** (`matriz.txt`).
2. **Construcción de su matriz básica (BM)**.
3. **Obtención de testores típicos** mediante el algoritmo YYC.
4. **Conversión de los testores típicos a una matriz B** (`matriz_B.txt`).
5. **Verificación de si B es una matriz básica**.
6. **Aplicación del algoritmo BT sobre la matriz B**.
7. **Visualización de típicos en formato de letras y binario**.
8. **Aplicación de operadores entre matrices**, como:
   - Concatenación generalizada  
   - Fusión combinatoria  
   - Operador diagonal generalizado  

Este proyecto está 100% automatizado y no requiere interacción del usuario, más que en en el Ejercicio 1 dónde el usuario debe escoger cantidad de filas y columnas.

Compilar ejercicios en C++. Se debe compilar en orden (1, 2, 3, 4, 5)
